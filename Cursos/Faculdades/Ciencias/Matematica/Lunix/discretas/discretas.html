* O que é?

      Do latim praetextus "cobrir ou tapar", são assuntos da matemática que estudam-se as "estruturas fixas e invariáveis da matemática" sendo esta a sua natureza que é oposta as "matemáticas contínuas".

* Escopo de Assuntos

    * Teoria dos Conjuntos *

      * É o assunto que estuda-se os "conjuntos" no universo, esta teoria foi desenvolvida pelo matemático alemão Georg Cantor(1845 - 1918) no final do século XIX através da necessidade de pesquisar mais acerca do finito e infinito e as propriedades dos números e a "cardinalidade" dos objetos de qualquer natureza nessas coleções.

      ** Conjuntos **

        * São coleções de objetos ou elementos no universo abstratos e concretos.

        * Características dos Conjuntos

          1. Elementos: É necessário que objetos façam parte do conjunto para ser considerado um, sejam eles abstratos ou concretos.

          2. Pertinência: Um elemento pertence "∈" ou não pertence "∉" a um conjunto.

          3. Definição: Um conjunto deve determinar de maneira clara se um objeto pertence ou não a ele.


        * Tipos de Conjuntos

        • Conjunto Vazio (∅): É o conjunto que não possui elementos ∅ = {};

        • Conjuntos Finitos: São conjuntos com um número finito de elementos F = {a, b, c};

        • Conjuntos Infinitos: São conjuntos com infinitos números de elementos I = {a, b, c, ...};

        • Conjunto Unitário: Um conjunto que contém apenas um elemento, U = {y};

        • Conjunto Universal: Um conjunto que contém todos os elementos de um conceito ou universo considerado, Área = {Subáreas};

        • Conjunto Subconjunto: Um conjunto cujos elementos pertencem a outro tipo de conjunto, C = {a, b, c} e B = {a, π, d} então C ⊆ B (C contém elementoem B);

        • Conjunto das Partes: Um conjunto de todos os subconjuntos de um determinado conjunto, Se C = {a, b, c}, C é P(C) = {∅}, {a}, {b}, {c}, {a, b, c};

        • Conjunto Igual: Dois ou mais conjuntos que possuem elementos em comum, C = {x, y, z} e D = {z, y, x} então C = D;

        • Conjunto Disjunto: Dois ou mais conjuntos que não possuem elementos em comum, A = {○, □} e B = {Δ, X}, então A ∩ B = ∅;

        • Conjunto Numérico: Conjuntos que contém os números divididos em categorias especifícas da Aritmética, N (números naturais), Z (números inteiros), Q (números racionais), R (números reais) e C (números complexos);
